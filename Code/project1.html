Home Automation and Monitoring System

I create a monitoring system using Node-RED and the MQTT protocol.
I use a Raspberry Pi as the central server (MQTT broker), where sensors
like LDR or a temperature sensor (DHT11) send their data. With Node-RED,
I build a visual interface to see this data in real-time and control actions,
like turning on or off a light or opening a door using servo motor.
It works like a conversation: sensors "talk" (publish data),the Raspberry Pi
"listens" (MQTT broker), and I, through the interface, decide what to do.

Environmental Monitoring Dashboard

This project uses a Raspberry Pi Pico with connected sensors
(DHT11, LDR, and water sensor) to measure environmental parameters:
temperature, humidity, light intensity, and water presence. Data is
continuously collected and stored in a CSV file for analysis. A Python-based
dashboard, built with Tkinter, visualizes these measurements through multiple
pages for efficient monitoring. The dashboard includes a home page
introducing the system, a display page with options for viewing
individual graphs for each sensor, a combined graph showing all parameters together,
and a table view for raw data. Additionally, a support page is provided for user
feedback, and a manual page guides users through the application and so much more.
